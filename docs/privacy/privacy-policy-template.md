## 4. How We Store Your Data

### 4.1 Storage Location

**Primary Storage:** European Union (GDPR Compliant)

All your personal data is stored in the **European Union** to ensure GDPR compliance:

| Data Type | Storage Service | Location |
|-----------|-----------------|----------|
| Account Data | Firebase Authentication | EU Multi-region |
| User Profiles | Cloud Firestore | `europe-west1` (Belgium) |
| Test Results | Cloud Firestore | `europe-west1` (Belgium) |
| AI-Generated Tests | Cloud Firestore | `europe-west1` (Belgium) |
| Application Code | Firebase App Hosting | `europe-west1` (Belgium) |

### 4.2 Data Security

We implement industry-standard security measures:

- ✅ **Encryption at Rest:** All data encrypted using AES-256
- ✅ **Encryption in Transit:** TLS 1.3 for all communications
- ✅ **Access Controls:** Role-based access with Firebase Security Rules
- ✅ **Authentication:** OAuth 2.0 via Firebase Authentication
- ✅ **Monitoring:** Real-time security monitoring and logging

### 4.3 Data Retention

| Data Type | Retention Period | Deletion Trigger |
|-----------|------------------|------------------|
| Active Account Data | Indefinite (while account active) | Account deletion |
| Test Results | Indefinite (while account active) | Account deletion |
| User Preferences | Indefinite (while account active) | Account deletion |
| Deleted Account Data | 30 days (backup retention) | Hard delete after 30 days |
| Application Logs | 90 days | Automatic deletion |
| Analytics Data | 26 months | Google Analytics auto-delete |

## 6. Your Rights Under GDPR

You have the following rights regarding your personal data:

### 6.1 Right to Access (Article 15)
You can request a copy of all personal data we hold about you.

**How to Exercise:** [Settings → Privacy → Download My Data] (TO BE IMPLEMENTED)

### 6.2 Right to Rectification (Article 16)
You can update or correct your personal data at any time.

**How to Exercise:** [Settings → Profile → Edit Information]

### 6.3 Right to Erasure ("Right to be Forgotten") (Article 17)
You can request deletion of your account and all associated data.

**How to Exercise:** [Settings → Privacy → Delete My Account] (TO BE IMPLEMENTED)

**Note:** Account deletion is permanent and cannot be undone. All your data will be deleted within 30 days.

### 6.4 Right to Restrict Processing (Article 18)
You can request to pause certain data processing activities.

**How to Exercise:** [Settings → Privacy → Manage Consents]

### 6.5 Right to Data Portability (Article 20)
You can export your data in a machine-readable format (JSON).

**How to Exercise:** [Settings → Privacy → Export My Data] (TO BE IMPLEMENTED)

### 6.6 Right to Object (Article 21)
You can object to certain types of data processing.

**How to Exercise:** [Settings → Privacy → Opt Out of Analytics]

### 6.7 Right to Withdraw Consent (Article 7)
You can withdraw consent for analytics, functional cookies at any time.

**How to Exercise:** [Settings → Privacy → Cookie Preferences]

### 6.8 Right to Lodge a Complaint
If you believe we are not complying with GDPR, you have the right to complain to a supervisory authority.

**EU Supervisory Authorities:** [https://edpb.europa.eu/about-edpb/about-edpb/members_en](https://edpb.europa.eu/about-edpb/about-edpb/members_en)
